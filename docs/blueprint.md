# **App Name**: Ayurveda at Tips

## Core Features:

- Landing Experience: Showcase the brand mission, tagline, and interactive elements including a hero section with the brand logo and a horizontal slideshow explaining key aspects like personalized dosha detection, daily Ayurvedic tips, and expert consultations. Features call-to-action buttons for starting the wellness journey or taking the dosha test.
- Secure User Authentication: Enable users to sign up or log in securely using Google or email, gating access to personalized features and storing user credentials via Firebase Authentication.
- Ayurvedic Dosha Quiz: A simple 5-question interactive quiz designed to determine the user's primary Dosha type (Vata, Pitta, or Kapha). Quiz results are saved to the user's profile in Firestore.
- Personalized Prakriti Profile: After completing the Dosha Quiz, users are presented with a profile dashboard displaying their dominant Dosha, a concise explanation of its characteristics, and initial lifestyle recommendations tailored to their unique constitution. Data is retrieved from Firestore.
- Personalized Dashboard & Daily Tips Tool: A dynamic user dashboard that greets the user by name, displays their Dosha information, and uses a generative AI tool to provide a personalized daily Ayurvedic tip (e.g., diet, yoga, meditation) based on their determined Dosha, stored and managed in Firestore.
- User Profile Management: Allow authenticated users to view and edit their basic profile details, such as name and identified Dosha type, which are persisted in Firestore.

## Style Guidelines:

- Primary color: Muted Olive Green (#A5A04F) to evoke an herbal, natural, and calming essence, contrasting gently with the light background.
- Background color: A highly desaturated, bright Creamy Yellow (#F9F8ED) with a subtle green tint, maintaining a serene and clean aesthetic. This visually complements the primary color while providing a light, calming canvas.
- Accent color: Earthy Brown (#935C39) for crucial interactive elements and to introduce visual depth, enhancing the natural and grounding feel of the palette. This offers significant contrast and complements the primary and background colors harmoniously.
- Headings font: 'Playfair Display' (serif) for a premium, elegant, and editorial feel, aligning with the wellness branding. Body text font: 'Montserrat' (sans-serif) for modern readability and a clean, minimalist touch.
- Use minimal and refined iconography, drawing inspiration from herbs, leaves, and traditional Ayurvedic tools to reinforce the brand identity without cluttering the UI. This contributes to the clean and nature-inspired design style.
- Adopt a mobile-first, responsive layout characterized by clean lines, rounded UI cards for content segmentation, and subtle soft shadows to provide depth without visual heaviness. The design prioritizes spaciousness and clarity.
- Incorporate smooth, fluid page transitions and subtle micro-animations to enhance user engagement and provide a seamless, premium app-like experience. Animations will be understated to maintain a calming atmosphere.
